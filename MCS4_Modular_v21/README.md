# MCS4 Modular v21 / Version 2.0

Änderungen:
- Version auf 2.0 gesetzt.
- Sensor-Konfiguration zeigt jetzt, ob ein Sensor in der aktuellen Sitzung erkannt wurde.
- Neuer Button **Ungenutzte Sensoren bereinigen** entfernt alte Konfigurationseinträge, die in der aktuellen Sitzung nicht mehr erkannt wurden.
- Hilfreich nach Decoder-/Page-Line-Änderungen, z. B. alte Einträge wie `5:3:0`, wenn aktuell nur `5:4:0` empfangen wird.

Hinweis:
Vor dem Bereinigen zuerst Simulator, Player oder RS422 laufen lassen, damit aktuelle Sensoren erkannt werden. Danach erst bereinigen.
