# MCS-4 Professional Monitor - Version 0.5

Stabile Basisversion mit:

- PySide6 GUI
- COM-Port-Erkennung
- Simulator
- Sensordashboard
- Trenddiagramm
- HEX-Telegrammmonitor
- Diagnoseanzeige
- thread-sicherem Event-Fluss ohne SQLite-Absturz

## Start

```cmd
.venv\Scripts\activate.bat
pip install -r requirements.txt
python main.py
```
