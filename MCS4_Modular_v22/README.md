# MCS4 Modular v22 / Version 2.1

Neue Funktionen:

- Neuer Tab **Protocol-Statistik**
- Live-Zählung aller Telegramme
- Telegramme pro Sekunde
- WordType-Verteilung
- häufigste Messpunkte / Nummern
- häufigste Data-Value-Keys (MP:Page:Line)
- Zähler für formal ungültige Telegramme

Diese Version dient als Live-Protocol-Analyzer für echte MCS-4-Aufzeichnungen und RS422-Livebetrieb.

Start:

```cmd
python main.py
```

Vor dem produktiven Einsatz bitte wie gewohnt den funktionierenden Stand mit Git sichern.
