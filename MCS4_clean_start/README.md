# MCS4 Monitor Clean Start

Saubere Neustart-Version. Keine Threads, keine Datenbank, kein Hintergrundzugriff auf GUI.

Start:

```cmd
.venv\Scripts\activate.bat
pip install -r requirements.txt
python main.py
```

Test: Modus `Simulator`, dann `Start` klicken. Sensordaten müssen im Dashboard erscheinen.
