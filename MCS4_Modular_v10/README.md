# MCS-4 Monitor Version 1.0

Version 1.0 erweitert Version 0.9 um einen Telegramm-Explorer.

## Neu

- Telegramm-Explorer Tab
- Zerlegung des letzten Telegramms in Sync, Header, Adressen, Channel, Einheit und Rohwert
- Decoder bleibt mit Simulator, Player und RS422 nutzbar

## Start

```cmd
python main.py
```

## Test

1. Simulator wählen
2. Start drücken
3. Tab **Telegramm-Explorer** öffnen
4. Dort wird das jeweils letzte dekodierte Telegramm erklärt
