# MCS4 Monitor

Professionelle Windows-Diagnosesoftware für MCS-4 RS422-Daten.

## Version 0.3

Diese Version enthält ein lauffähiges Dashboard mit Simulator, Trenddiagramm, Alarmübersicht und COM-Port-Grundlage.

## Start

```cmd
.venv\Scripts\activate.bat
python main.py
```

## Test ohne Hardware

1. Modus auf `Simulator` lassen.
2. `Start` klicken.
3. Dashboard, Telegramme, Alarme und Diagnose beobachten.
