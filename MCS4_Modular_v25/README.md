# MCS-4 Professional Monitor - Version 2.3.1

Customer-ready English UI base with developer helper scripts.

## Start from source
Double-click:

`start_dev.bat`

or in CMD:

```cmd
start_dev.bat
```

## Install or update dependencies

```cmd
install_requirements.bat
```

## Build EXE

```cmd
build_dev.bat
```

Output:

`dist\MCS4_Monitor\MCS4_Monitor.exe`

## Clean build/cache files

```cmd
clean_build.bat
```

## Check environment

```cmd
check_environment.bat
```

## Important folders

- `recordings` - MCS log files
- `exports` - CSV, Excel, PDF, PNG exports
- `logs` - future log files
- `config` - future configuration files
