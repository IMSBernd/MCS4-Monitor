# MCS-4 Monitor Version 0.9

Version 0.9 erweitert die stabile Basis um eine modernere Kopfzeile mit blinkender REC-Anzeige, Aufnahmedauer, Telegrammzähler und Dateianzeige.

## Start

```cmd
python main.py
```

## Test

1. Simulator wählen
2. Start drücken
3. Aufnahme Start drücken
4. REC-Anzeige blinkt und zeigt Dauer sowie Telegrammzähler
5. Aufnahme Stop drücken
6. Player-Datei laden und im Player-Modus wiedergeben

## Abhängigkeiten

```cmd
pip install PySide6 pyserial pyqtgraph
```
